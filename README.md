# EduTask - 교사용 올인원 매니저 🏫

EduTask는 현직 교사 및 강사분들의 학급 운영, 수업 기록, 학생 및 학부모 관리를 혁신적으로 돕기 위해 기획된 **교사용 올인원 매니저**입니다.

## 🌟 주요 기능 (Core Features)

- **반별 관리(Class CRUD)**: 학급별(예: 1학년 1반, A반 등) 그룹 생성/수정/삭제 및 반별 학생 매칭
- **학생 관리(CRUD)**: 반별 학생 사진, 성향, 연락처 등록/수정/삭제 및 출결/지각 체크
- **수업 매니저**: 반별 교재명, 페이지 수, 진도 범위, 학생별 5단계 이해도 및 오답 유형 기록
- **🤖 교재 마의 구간 알림**: 학생들의 이해도가 낮은 특정 페이지 구간을 AI가 분석해 보충 자료 필요성 알림
- **🤖 상담 치트키**: 상담 키워드 기반 AI 상담 요약 및 학부모용 정중한 문자 자동 변환
- **👑 생기부 빌더(Premium)**: 평소 수업/상담 기록을 바탕으로 생활기록부 문구 AI 초안 생성
- **행정/보안**: 공문 데드라인 우선순위 알람, 관리자/교사 권한 분리, 생체 인증/비밀번호 잠금
- **데이터 활용**: 모든 기록을 엑셀(Excel) 및 PDF로 내보내기, 멀티 디바이스 동기화 및 클라우드 백업
- **수업 도구**: 랜덤 발표자 뽑기, 포모도로 타이머 위젯, 자주 쓰는 수업/상담 템플릿
- **📈 학생 성장 리포트**: 5단계 이해도 데이터를 시각화한 그래프 생성 및 엑셀 출력 연동
- **📄 공문 스캔**: 행정 업무 알람에 이미지/PDF 파일을 첨부하여 모바일에서 바로 확인하는 기능
