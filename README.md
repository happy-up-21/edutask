# EduTask - 교사용 올인원 매니저 🏫

EduTask는 현직 교사 및 강사분들의 학급 운영, 수업 기록, 학생 및 학부모 관리를 혁신적으로 돕기 위해 기획된 **교사용 올인원 매니저**입니다.

## 🌟 주요 기능 (Core Features)

- **반별 & 학생 관리**: 학급 생성/수정/삭제 및 학생 정보(성향, 연락처) CRUD
- **수업 매니저**: 교재명, 페이지, 진도, 5단계 이해도 및 오답 유형 텍스트 기록
- **AI 분석**: 교재 마의 구간 알림, 상담 키워드 기반 문자 변환, 생기부 초안 생성
- **데이터 활용**: 기록된 모든 데이터를 엑셀(Excel) 및 PDF로 내보내기
- **보안 및 로그인**: 관리자/교사 권한 분리, 비밀번호 및 생체 인증 기반 로그인 시스템 지원
- **유틸리티**: 행정 업무 데드라인 우선순위 알람, 랜덤 발표자, 타이머 위젯
